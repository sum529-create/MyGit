* 쿠키
1. 저장하기 
서버가 만들어서 웹브라우저에 보관.
쿠키는 자바로 만듬 -> server
그걸 가지고 작업했다 -> web browser
서버가 만들어서 클라이언트에게 쿠키를 전달하기 위해 response을 사용한다.
서버가 클라이언트에게 전달하면 클라이언트가 쿠키를 저장시킨다.
response.addCookie(쿠키명)

2. 불러오기
있나 없나 검사한다.
서버는 쿠키를 request의 header에 담아서 가져온다.
서버가 만든것은 request
하나만 읽을 수 없고 다 같이 모든 쿠키를 읽어온다.
